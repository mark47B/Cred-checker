# Credentials-checker
Небольшое .NET приложение для проверки Логина и Пароля

## Run ✈️

- Разархивировать zip файл

### В директории с проектом
- Собираем проект

```bash
    dotnet build
```

- Запускаем

```bash
  dotnet run
```

Url, порт и urn прописаны в FE_test/main.js в переменной URL ('http://localhost:5211/api/Access')
